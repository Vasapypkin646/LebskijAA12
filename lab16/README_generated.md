## 📝 Документация проекта: checkers.py

### 📌 Описание проекта

Этот проект представляет набор утилит Python, направленных на решение повседневных задач обработки данных и оптимизации работы приложений.

#### ✅ Основные возможности:
1. **Проверка валидности email** — проверка корректности введённого email по регулярному выражению.
2. **Сортировка словарей по ключу** — гибкая сортировка списка словарей по выбранному ключу.
3. **Декорирование функции таймером** — подсчёт времени выполнения любой функции, удобной обёрткой для профилирования производительности приложения.

---

### 🛠 Установка зависимостей

Все необходимые зависимости уже встроены в код и импортируются автоматически внутри скрипта. Поэтому установка дополнительных пакетов не требуется.

### 💻 Примеры использования

#### 🔍 Пример №1: Проверка валидности email

```python
from checkers import validate_email

# Пример корректного email
email_valid = validate_email('test@example.com')
print(email_valid)  # Вернёт True

# Пример некорректного email
email_invalid = validate_email('invalid.email')
print(email_invalid)  # Вернёт строку ошибки
```

#### 🔍 Пример №2: Сортировка словарей по ключу

```python
from checkers import sort_by_key

data = [
    {'id': 5, 'name': 'John'},
    {'id': 3, 'name': 'Alice'},
    {'id': 7, 'name': 'Bob'}
]

sorted_data = sort_by_key(data, 'id', reverse=True)
print(sorted_data)
```

Результат будет выглядеть следующим образом:
```
[
  {'id': 7, 'name': 'Bob'},
  {'id': 5, 'name': 'John'},
  {'id': 3, 'name': 'Alice'}
]
```

#### 🔍 Пример №3: Измерение времени выполнения функции

```python
from checkers import timer

@timer
def example_function():
    time.sleep(1)

example_function()
```

После запуска выведется информация о времени выполнения этой функции.

### 📚 Описание функций

#### 🎯 `validate_email(email)`

Функция проверяет корректность email по шаблону регулярного выражения.

**Параметры:**
- `email` (строка): email адрес для проверки.

**Возвращает:**
- `True`, если email корректный.
- Сообщение об ошибке, если email некорректный.

#### 🏆 `sort_by_key(data, key, reverse=False)`

Функция осуществляет сортировку списка словарей по определённому ключу.

**Параметры:**
- `data` (список словарей).
- `key` (строка): ключ по которому производится сортировка.
- `reverse` (логический флаг, опционально): определяет порядок сортировки (от меньшего к большему).

**Возвращает:**
Отсортированный список словарей.

#### 🕰️ `timer(func)`

Декоратор, который вычисляет время выполнения декорируемой функции.

**Параметры:**
- `func`: функция, которую нужно замерить.

**Возвращает:**
Обвёрнутую функцию с добавлением функционала замера времени выполнения.
